# ¡Hola! Bienvenidos a mi espacio de desarrollo SENA 👋

Este repositorio está dedicado exclusivamente al diseño, documentación y código de mi proyecto formativo para el tecnólogo en **Análisis y Desarrollo de Software (ADSO)** bajo los lineamientos y requerimientos del **SENA**.

---

## 🚀 El Proyecto: EIEN-System
Este proyecto es un sistema de gestión logística de alta precisión para Kizuna Logística S.A.S. Desarrollado bajo estándares de eficiencia Japonesa. (Kizuna Logísitca S.A.S es una empresa de logísitca ficticia creada para fines académicos). 

### 🛠️ Tecnologías y Metodologías Aplicadas en el SENA:
* **Metodología:** SCRUM / Ágil.
* **Backend:** Python, FastAPI.
* **Frontend:** HTML5, CSS3, TypeScript, React.
* **Base de Datos:** PostgreSQL.
* **Documentación:** Fases de Análisis, Diseño, Desarrollo e Implantación.
